SnipForMammal

Track History
	Displays list of tracks that have been played. You can copy the name of the song or the direct link to Spotify.
	
Settings
	Allows you to customize settings in the program such as how often the program updates information, or the format of the text sent to the text file.
	
Custom Text
	Allows you to manually enter any text you would like to display. This text is overriden when a new song is detected playing.
	
Force Update
	Forces an update to check for a song playing.
	
Show Debug Console
	Shows the debug console containing useful information if something goes wrong.
